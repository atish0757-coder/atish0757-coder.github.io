How I Like to Learn New Tools

Here is an ordered list with an unordered list inside it (requirement check ✅).

  1. Start with the big picture
    • What problem does this tool solve?
    • Where is it used in real companies?
  2. Explore hands-on tutorials
    • Recreate dashboards and financial models
    • Break things and figure out how to fix them
  3. Apply the tool to my own questions
    • Investment and valuation questions
    • Economic development in Nepal
